# Gym Community App

Workout tracking and gym community platform for Iron Arms Gym

## Backend Setup
```bash
cd backend
uv sync
cp .env.example .env   # Then fill in real values
uv run uvicorn app.app:app --reload
```

API Docs: http://localhost:8000/docs
